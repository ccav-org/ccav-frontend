# CCAV.com 首页 V1 后续修正清单

> 当前首页核心内容已经齐全。后续不要再大幅加模块，重点做适配、素材替换、层级精修和官方口径校准。

## 高优先级

- [ ] 首屏 CTA 继续收敛与命名确认
  - 当前：`预约体验课` / `查看课程体系`
  - 待确认是否改为：`预约免费体验课`
  - 合作入口保留在导航 `合作加盟` 和后续合作模块，不回到首屏。

- [ ] 团体标准官方口径最终确认
  - 当前首页只保留精简表达：依据《人工智能视频制作人员要求》团体标准。
  - 待回国后确认正式写法，包括：
    - 中华文化促进会发布
    - 起草、论证、公开征求意见、专家审定
    - 全国团体标准信息平台备案
    - 社会艺术水平考级委员会、动漫艺术专家委员会等参与关系
  - 不提前写死证书发放主体、授权关系和外部背书。

- [ ] 证书体系文案与样张校准
  - 当前：首页展示三级能力证书、三级教师证书。
  - 待确认：
    - 证书正式名称
    - 证书等级命名
    - 培训师 / 指导教师是否分别也分初中高
    - 电子证书与纸质证书样式
    - 查询、防伪、编号规则

- [x] 移动端首屏高度和信息密度再压缩
  - 当前移动端首屏内容偏长。
  - 重点检查：使命条、主视觉、按钮、三步路径、证书模块在手机上的节奏。

## 中优先级

- [x] 数据背书前置
  - 当前数据条仍靠后。
  - 可考虑移到首屏或证书模块之后，形成早期信任背书。
  - 数据文案需避免硬凑和重复。

- [x] 三步路径视觉增强
  - 当前：`学方法 / 做作品 / 获取证书`
  - 待优化为更明确的流程线、箭头或进度关系。
  - 不要再和六大业务板块重复。

- [ ] 专家与师资素材统一
  - 当前远程专家照片风格不统一。
  - 待替换为统一尺寸、统一背景或统一裁切的正式素材。
  - 首页只展示代表专家，详细介绍放师资页。

- [x] 适合谁学习模块重构
  - 当前六类受众平铺，信息偏散。
  - 后续可改为三类：
    - 学生
    - 创作者 / 职场人
    - 机构 / 学校
  - 每类强调具体学习收益。

- [x] 作品展示增强
  - 当前改为官方初 / 中 / 高三级示范作品，不再使用播放按钮暗示已有视频。
  - 官方示范作品需要展示创作过程：提示词拆解、迭代记录、关键帧 / 分镜 / 视频制作流程。
  - 学生作品上线后，再逐步替换为真实学员作品、项目视频或前后对比。
  - 后续补齐三级示范视频，并增加更完整的过程页或弹层。

## 低优先级

- [ ] 全国教学网点模块升级
  - 当前为示意图。
  - 后续有真实网点数据后，再考虑城市搜索、省份列表或真实地图。

- [x] 底部行动入口收敛
  - 当前底部入口较平均。
  - 后续可分为：
    - 个人学习：预约体验 / 获取资料
    - 机构合作：咨询合作 / 申请网点

- [ ] 品牌色和图片风格统一
  - 保持科技蓝为主、青绿为辅。
  - 不引入过多霓虹、橙色、紫粉等强营销色。
  - 替换通用素材图，逐步改为 AI 视频创作真实场景。

- [ ] 首页截图和上线前视觉 QA
  - 每次重要调整后重新保存桌面截图。
  - 检查桌面、平板、手机三档布局。
  - 检查文字换行、按钮溢出、卡片高度、图片裁切。
